# chatserver
基于C++ Muduo网络库实现的集群聊天服务器与客户端，支持Nginx TCP负载均衡，使用Redis实现跨服务器消息通信，MySQL持久化存储数据

编译方式
cd build
re -rf *
cmake ..
make

项目介绍
本项目是一款C++ 分布式集群聊天系统，服务端基于高性能 Muduo 网络库开发，通过 Nginx 实现 TCP 层负载均衡，支持多服务器节点横向扩展；
使用 Redis 发布/订阅机制实现跨服务器消息转发，MySQL 完成用户信息、离线消息、聊天记录的持久化存储；
配套实现控制台客户端，支持一对一聊天、群聊、在线状态管理等功能。

核心技术栈
网络库：C++ Muduo（Reactor模型，高并发网络IO）
负载均衡：Nginx TCP 负载均衡（集群分发请求）
消息中间件：Redis Pub/Sub（跨服务器消息通信）
数据库：MySQL（用户/离线消息/群组数据持久化）
序列化：JSON（前后端/服务间数据交互）
开发语言：C++11

核心功能
1. 用户注册、登录、退出
2. 一对一聊天（同服务器/跨服务器）
3. 群组聊天（创建群组、加入群组、群消息转发）
4. 离线消息存储与推送
5. 在线状态实时管理
6. 多服务器集群部署
7. Nginx 负载均衡分发客户端请求

项目亮点
1. 高并发：基于 Muduo 异步事件驱动模型，支撑海量客户端连接
2. 可扩展：支持集群部署，通过 Nginx 轻松扩容服务器节点
3. 跨服通信：Redis 解耦集群节点，实现跨服务器消息实时转发
4. 数据可靠：MySQL 持久化所有业务数据，离线消息不丢失
5. 生产级：模块化设计，代码清晰，可直接部署运行

部署环境
- Linux 环境（Ubuntu/CentOS）
- Muduo 网络库
- Redis 服务
- MySQL 服务
- Nginx（配置TCP负载均衡）
